# Bus Tracking System - Project Report

## Project Overview
This is a React-based Bus Tracking System application built using Vite as the build tool. The application allows users to search for buses by their ID and view their current location, driver information, and status on a map.

## Technology Stack
- **Frontend Framework**: React 19.2.4
- **Build Tool**: Vite 8.0.0
- **Linting**: ESLint with React-specific rules
- **Styling**: CSS (main.css)
- **Data**: Static JavaScript array (buses.js)
